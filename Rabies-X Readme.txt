Rabies-X Readme
---------------

Controls:
=========

"Todo" Text: A, D, S, W

Test Model: Left, Right, Up, Down, or (A, D, S, W)

Health Bar: PageUp, PageDown

Camera Spring System: Spacebar (enable/disable)

Camera Spring Constant:  +, - (increase/decrease)

Fullscreen: Alt and Enter (on/off)

Help: H (hide/show)
